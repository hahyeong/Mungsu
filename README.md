# Mungsu
이화여자대학교 컴퓨터공학과 2022-1 캡스톤디자인프로젝트B 그로쓰 28팀의 프로젝트 레포지토리입니다.
- 프로젝트 상세 설명 및 시연 영상: https://youtu.be/cbvxrYYcmH0

### 프로젝트 개요
펫 CCTV를 이용한 반려견 건강 진단 서비스 <멍수무강>은 <br>
딥러닝 기술을 기반으로 반려견의 행동을 실시간으로 분석하고 이를 건강 진단을 위한 데이터화 합니다. <br>
생성된 데이터를 바탕으로 반려견의 건강을 진단하여 리포트 형식으로 이용자에게 보여지게 됩니다. <br>
또한, 반려견의 이상행동이 포착되는 경우 이용자에게 이를 실시간으로 알려주어 위험을 예방합니다.
